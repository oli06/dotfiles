# .zshrc
1. Add `export XDG_CONFIG_HOME="~/dotfiles" to your ~/.zshrc file
2. Add `source "$XDG_CONFIG_HOME/.zshrc"` to your ~/.zshrc file
3. Restart zsh or source .zshrc file
